## Overview

This repository contains the source code for **www.allistairlee.com**. Built with 11ty, the project emphasizes simplicity, performance, and modern web development best practices. The site’s content, templates, and assets are all maintained here.

## Features

- **Static Site Generation:** Powered by 11ty for fast, SEO-friendly static pages.
- **Flexible Templating:** Customize your pages with your preferred templating language (Nunjucks, Liquid, etc.).
- **Responsive Design:** Optimized for all devices and screen sizes.
- **Performance Optimized:** Lightweight and efficient, ensuring quick load times.
- **Easy Maintenance:** Simplified content updates and code management.

## Technologies

- **[11ty (Eleventy)](https://www.11ty.dev/):** The static site generator powering the build.
- **[Node.js](https://nodejs.org/):** The runtime environment.
- **[Tailwind CSS](https://tailwindcss.com/):** For styling and interactivity.
- **Templating Language:** Nunjucks used for building dynamic templates.

## Deployment

The site is currently deployed via [Sevalla](https://sevalla.com).
